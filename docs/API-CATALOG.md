# 공단·정부 API 사전 (2026-07-17 정리)

이 앱에서 쓸 수 있는 외부 API 전체 목록. 키는 **딱 2종**이다:
플랫폼 공용 조회키(공공데이터포털, 1개) + 업체별 민원24 키(보고용).

## ① 공개 조회 API — 공공데이터포털 (플랫폼 공용 키 1개, 서버 프록시로 호출)

| API | 뭘 주나 | 조회 기준 | 우리 사용처 |
|---|---|---|---|
| **건물별 승강기 정보** | 건물의 전체 호기 제원(종류·형식·설치일·정원) + 검사유효기간 + 판정결과 | 고유번호 1개 → 같은 건물 전체 반환 | ✅ `/api/elevator-info` — 현장 자동등록, 호기 상세 |
| **검사이력→부적합 상세** (InspectsafeService) | 검사 회차별 부적합 항목·검사원 의견 | 고유번호 + 기준일 | ✅ `/api/elevator-fail-detail` — 조건부/불합격 상세 |
| 승강기설치정보 조회 | 설치일자·형식·정격속도 | 고유번호 또는 자료생성일 | 미사용 (건물별 API와 중복) |
| 승강기 정보 및 검사이력 | 지역·건물 기준 승강기 목록+이력 | 시도·시군구·건물명 | 미사용 |
| **사고·고장 승강기 이력** | 사고/고장 이력 | 지역·건물 | 미사용 — 고장이력 참고자료 후보 |
| 건물별 승강기 운행정보 | 운행 유효기간 등 | 고유번호 | 미사용 |
| 유지관리업체 정보 조회 | 지역별 업체 목록(회사명·전화·보수범위) | 지역 | 미사용 — **회사 단위만, 소속 기사 없음** |
| (국세청) 사업자 상태조회·진위확인 | 계속사업자/휴폐업, 대표자 대조 | 사업자번호 | 가입 검증 예정 — **기존 키에 활용신청만 추가** |

**파일데이터 (연 1회 CSV):** 승강기 설치 현황(전국 86만 대, **유지관리업체 컬럼 포함** → 가입 시 업체명 역매칭 자산), 승강기 검사 결과, 유지관리업체 현황.

## ② 업체 인증 API — 승강기민원24 (업체당 인증키+암호키, 유지관리업체만 발급)

| API | 뭘 하나 | 비고 |
|---|---|---|
| **자체점검결과 등록** (RegistInspectionService) | 월 자체점검 결과를 공단에 제출 | 전문에 점검자 ID(SELCHK_USID=profiles.minwon_id)·점검항목코드 필수, MD5 서명 |
| 자체점검결과 삭제 | 제출 건 삭제 | COMPANY_UNIQUE_NO(우리가 부여한 고유값) 기준 |

## ③ 없는 것 (자주 묻는 것 — 찾지 말 것)

- ❌ 업체명으로 승강기 검색 → 연간 CSV로 대체 (역매칭)
- ❌ 주소·건물명으로 검색 (건물별 API는 고유번호만 — 실험으로 확인)
- ❌ 소속 기술인력/회원 정보 (개인정보 — 업체계정 화면에서만)
- ❌ 자체점검 결과 **조회** (등록·삭제뿐 → 공단 기록과 월별 자동 대조 불가)
- ❌ 검사 신청·예약
