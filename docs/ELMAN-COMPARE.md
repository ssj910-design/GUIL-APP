## 엘맨PRO 전체 기능 목록 (메모)

**9대 모듈**: 인사(입퇴사·발령·부서·근태·급여·자격) / 거래처(CMS·지로·콜센터연동) / 회계(매출입·수금미수·전자세금계산서 홈택스·급여) / 유지보수(현장·승강기·고장·점검·검사·견적·공사) / 설치(수주·견적·설계내역서·도면) / 자재(제품코드·발주·입고·불출·재고·송장) / 고객만족(민원·안심문자·만족도평가) / 통계(고장분석·매출·장기미수·점검완료율·연마감) / 전자결재(13종 문서·기안·지출결의서)

**차별 기능**:
- 공단 연동: 검사 매일 자동갱신+아침7시 문자, 점검계획 자동생성+점검일지 자동전송(당일 저녁)+재전송, 호기정보 실시간(타사 현장도), NFC 점검
- GPS: 기사 실시간 위치 지도에서 고장 배정, 현장 담당자 배치
- 계좌 연동: 은행 거래내역 15분 내 자동 수집, 수금 대사
- 고객 안심: 고장 시 고객에게 웹링크(기사 위치·연락처·도착 ETA 지도) + 처리 후 5점 만족도 평가
- 콜센터: CID 수신전화 → 거래처·현장·최근이력 팝업, 통화기록, 근거리 기사 표시
- 전광판: 사무실 TV 상황판 (실시간 고장 접수/배정/처리, 점검 현황, 아침조회·일정·캘린더) + **고장 예측 (TensorFlow, 매일 새벽)**
- FIX HELPER: 기종별 오류코드 DB + 선배 경험 기록 (수동 입력 DB)
- 견적서: 자재 쇼핑담기 → 결재승인 → 이메일 전송 → 발주 연동
- 현장 서명(점검자/부점검자/고객) → PDF 이메일 즉시 전송
- 사진 대지(보드판 부착), 공사 전중후 사진 → 확인서 출력
- 원격 지원: 현장↔사무실 영상+음성, 붉은 원 지시, 녹화
- 원격 관제(NEW): 삼일엘텍 SICON 4000+ 연동 — 실시간 상태·원격 제어·자동 고장 접수·운행 리포트(운행시간/거리/도어개폐 통계)
- 창고 관리(옵션): IoT 도어락 + CCTV + 입출고
- 카카오 네비 원터치, 월마감/년마감 회의 화면, 첫화면 팀별 그래프
- 우수업체 선정 컨설팅(별도 상품)
- VISION(미래): 은행 계좌 연동 확대, **채팅**, 자동작도, 발주서 자동 제작, 호기별 고장 예측, **웹 ERP**

## 우리(GUIL-APP)와 비교

### ✅ 우리도 있음
| 엘맨 | 우리 |
|---|---|
| 유지보수 현장·승강기 관리 | 현장관리 + units (공단 엑셀 일괄 등록은 우리가 더 나음) |
| 고장 접수>배정>처리 | 고장접수 탭 + 관리자 고장 관제 |
| 검사 일정·공단 호기정보 | 실시간 검사 관제 + /api/elevator-info (동일 컨셉) |
| 점검 계획(출석부) | 자체점검 self_checks |
| 견적·자재 신청 | 자재·견적 탭 (기사 신청 중심) |
| 공사·부품 사진 | 청구·자재 사진 업로드 |
| 통계 | 통계 메뉴 + **전국 비교(엘맨에 없음)** |
| 기사 실적·담당별 현황 | 기사 실적 테이블 (기초) |
| (VISION) 채팅 | **우리방 — 멘션·@모두·답장·좋아요·사진영상·갇힘사고 자동알림 (이미 보유)** |
| (VISION) 웹 ERP | 이미 웹(PWA) |

### ❌ 우리에게 없음 (엘맨 보유)
**단기에 따라잡을 것 (로드맵과 일치):**
1. 자체점검 공단 자동 전송 — 진행 중 (민원24 암호키 대기)
2. 고장 예측·FIX HELPER — **RAG 초석 심어둠, AI 검색으로 상위호환 가능** (엘맨은 수동 DB+TensorFlow)
3. 고객 안심 웹링크(ETA 지도)·만족도 평가 — 문자 인프라 계약 후
4. GPS 기사 위치·네비 연동 — T맵 API 키 받아둠
5. 전광판(사무실 상황판) — 관리자 콘솔 TV 모드로 저비용 구현 가능
6. 점검 서명 + PDF 이메일

**중장기 (SaaS 단계):**
7. 회계 전체 — 전자세금계산서(홈택스)·수금미수·지로/CMS·은행 계좌 연동·장기미수 자동 문자 (세금계산서는 가입 설계에 이미 반영)
8. 인사 ERP(급여·근태·발령) / 전자결재(13종 문서)
9. 설치(INSTALL) 부문 — 수주·설계내역서·도면
10. 자재 ERP 전체 — 발주·입고·재고·송장 (우리는 상비부품 kit_stock만)
11. 민원 관리 트랙, 콜센터 CID
12. 원격 관제(하드웨어 파트너십 필요)·원격 지원 영상통화·IoT 창고

### 🏆 우리만 있음
- 전국 비교 통계 (공단 파일데이터 88.5만대 벤치마크)
- 팀 채팅 완성형 (엘맨은 계획 단계)
- 공단 엑셀 3분 온보딩 + 역매칭 온보딩 설계
- 웹 기반 — 설치·업데이트 불필요
- (진행 중) RAG 기반 AI — 빠른답변·유사고장 검색

**포지셔닝 시사점**: 엘맨은 "회계까지 포함한 무거운 설치형 ERP", 우리는 "현장·공단연동 중심의 가벼운 웹앱 + AI". 엑셀·수기 업체를 뺏는 싸움에서는 온보딩 속도와 가격이 무기, 엘맨 고객을 뺏으려면 회계(세금계산서·수금)가 최소 요건.
